명령어 레지스트리

/start [업체명]
재료 보고 + 스타일 추천 → 컨펌 대기
/build [업체명]
구조 설계 보고 → 컨펌 후 코드 작성
/check
코드 오류 / 금지 패턴 / 구조 일치 검수
/scope [요청]
대형 작업 범위 확인 + 옵션 3단계 제시
/plan [작업명]
코드 작성 전 구조 설계 컨펌 게이트
/status
진행 중 프로젝트 현황 대시보드
/resume [프로젝트]
이전 작업 맥락 복원
/handoff [A→B]
아크 검문 → 인계 문서 생성
/review
결과물 전체 검토 + 렌즈 검수
/retro [프로젝트]
완료 회고 + 누적 로그 전체 출력
/log [내용]
실수·수정 기록
/mat [업체명]
재료 상태 확인 및 추적
